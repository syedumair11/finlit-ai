# 💰 FinShiksha: AI Powered Financial Wisdom

🚀 **An AI-driven financial literacy platform** designed to help users make smarter financial decisions using intelligent insights and machine learning.

---

## 📌 Problem Statement

Many individuals lack financial awareness and struggle with:

* Poor budgeting decisions
* Lack of investment knowledge
* No personalized financial guidance

---

## 💡 Solution

FinShiksha provides:

* 📊 AI-driven financial insights
* 🤖 Smart recommendations based on user input
* 📈 Data-driven financial decision support

---

## 🧠 Key Features

* 🔹 Personalized financial recommendations
* 🔹 AI/ML-based analysis
* 🔹 User-friendly interface
* 🔹 Data-driven insights

---

## 🏗️ System Architecture

User Input → Data Processing → ML Model → Insights Generation → Output (UI/API)

## ▶️ How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/syedumair11/finlit-ai.git
cd finlit-ai
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run backend

```bash
cd backend
python app.py
```
